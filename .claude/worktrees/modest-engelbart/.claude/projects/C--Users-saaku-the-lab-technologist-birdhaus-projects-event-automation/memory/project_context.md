The event automation app is one of the main operational moments for Birdhaus -- it's how classes get published to the website.

**Why:** Birdhaus is a rope/BDSM education venue that runs regular classes. The scheduling data lives in Google Sheets (maintained by operations), and this app automates pushing those events to the Wix website.

**How to apply:** Changes to this app directly affect the business's ability to publish classes. Treat it as production-critical. When making changes, prioritize reliability and clarity over cleverness.
